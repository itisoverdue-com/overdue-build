# Agent Instructions

## Agent skills

### Issue tracker

Issues and PRDs for this repo are tracked in GitHub Issues. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default Matt Pocock skills triage label vocabulary, including `ready-for-agent` for AFK-ready work. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context documentation layout with root `CONTEXT.md` and ADRs under `docs/adr/` when present. See `docs/agents/domain.md`.
